Welcome to my Spring Project Management App!

This application is designed to help teams manage their projects efficiently and effectively.

Technologies Used:

  • Spring Framework
  • Spring Boot
  • Spring Security (including OAuth2 login)
  • Spring Data JPA (with MySQL/PostgreSQL)
  • Spring MVC
  • Spring Session
  • Thymeleaf Templates
  • Bootstrap 5 HTML/CSS/Javascript

Purpose:

This app allows teams to collaborate on projects, track tasks, and streamline workflow.

How to Use:

To learn more about the features, please check the About section below.

Get Started:

Log in or sign up to start managing your projects! You can also sign in as a guest or use OAuth2 for easy access.
